# Voptix
Online real-time voxel renderer   

### Demo:
https://mateusztk.github.io/Voptix/   

### Features:
- [x] Real-time voxel rendering
- [x] Fully raytraced
    - [x] Primary rays
    - [x] Reflections
    - [x] Global illumination
    - [x] Soft shadows
- [x] Editable world
- [x] Chunk system
